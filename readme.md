## Ambient_player

---

### 배포 링크

[https://ambient-player-client.vercel.app/](https://ambient-player-client.vercel.app/)

<br />

### 컨셉

색상별로 연관되는 이미지와 소리를 제공합니다!

<br />

### 개발 목표

useRef를 활용하여 컴포넌트 제어하기!
로티 라이브러리를 활용하여 에니메이션 효과 극대화 하기!

<br />

### 기능 리스트

- [x] 애니메이션 및 소리 적용
- [x] 역동적 카드(팬딩) 컴포넌트 구현
- [x] 재생 및 중지 버튼 구현
- [x] 컴포넌트 클릭시 애니메이션 재생
